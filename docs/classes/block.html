Class Block

The purpose of Block is to specify the location and size of an element.

There are two ways to manipulate a blocks position and size. The first is a css like block specification - left, top, right, bottom, width, and height all with optional units. The second is just updating its result rect in pixels. The two are automatically kept in sync.
