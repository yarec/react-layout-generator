Enumeration Unit

Defines the units of location and size passed to data-layout or in IDataLayout. Most computation will be done in pixels.

Percent uses the container size as its basis.

Viewport uses the document's viewport as it's basis.

Preserve is similar viewport except that it uses the container size rather than the viewport as its basis.

Both Viewport and Preserve units will keep a square as a square. Percent will only render a square if its basis is also square.
